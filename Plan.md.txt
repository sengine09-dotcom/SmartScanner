**Platform:** Windows 11  
**Framework:** .NET 8.0 (WPF)  
**Language:** C#  
**IDE:** Visual Studio 2022

---

## 1. 🎯 Project Overview
A desktop application designed for Windows 11 to streamline the "Scan-to-Email" workflow. The app interfaces with scanner drivers (TWAIN/WIA), captures documents, converts them into a consolidated PDF file, and dispatches them via email based on user-defined parameters.

---

## 2. 🏗️ System Architecture
The application follows a modular structure to ensure maintainability:
1.  **UI Module (XAML):** A modern Windows 11 interface for user input and process monitoring.
2.  **Scanning Module:** Handles communication with hardware drivers to acquire raw image data.
3.  **PDF Engine:** Processes image buffers into a professional, compressed PDF document.
4.  **Mail Dispatcher:** Manages SMTP authentication and secure file transmission.

---

## 3. 🛠️ Tech Stack & Libraries
| Function | Library / Technology | Key Benefit |
| :--- | :--- | :--- |
| **GUI** | WPF (.NET 8) | Native Windows 11 look, Fluent Design support. |
| **Scanning** | `NTWAIN` or `WIA` | Wide compatibility with flatbed and ADF scanners. |
| **PDF Creation** | `QuestPDF` | High-performance, code-based PDF generation. |
| **Email Service** | `MailKit` | Robust, supports modern security protocols (TLS). |
| **Storage** | `System.IO` | Manages local temp files and naming conventions. |

---
